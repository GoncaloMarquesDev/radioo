# Radiofy

Radioo.FM é uma aplicação web de rádios online desenvolvida com **React e TypeScript**.

A aplicação permite aos utilizadores descobrir e ouvir estações de rádio de todo o mundo de forma simples e intuitiva.

## Funcionalidades

- Pesquisa de estações de rádio
- Player global para ouvir rádio
- Interface responsiva (mobile-first)
- Favoritos guardados localmente

## Tecnologias

- React
- TypeScript
- Vite
- Radio Browser API

## Como correr o projeto

```bash
npm install
npm run dev
```
